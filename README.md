DEPENDENCIES
============
IT++ (Version: 4.2-4)
GNU Radio (with UHD)

INSTALLATION
============
sudo apt-get install libitpp-dev
git clone git://github.com/bastibl/gr-ieee802-11.git
cd gr-ieee802-11
mkdir build
cd build
cmake ..
make
sudo make install

USAGE
===========
Open examples/rx.grc.
